2023 - 2024

Project MoodMatcher

O projeto desenvolvido utiliza uma API que gera frases com base na natureza selecionada, como saudabilidade, humor ou tristeza. Em seguida, uma imagem compatível com a mensagem é gerada, proporcionando uma combinação visual e textual harmoniosa. Esse recurso oferece uma experiência interativa e personalizada para os usuários, que podem selecionar o tipo de frase desejada e receber uma representação visual correspondente, tornando a comunicação mais expressiva e envolvente.

Sistema de Gerenciamento de Biblioteca com Otimização e Backup

O projeto integrou CRUD otimizado com reutilização eficiente de espaço, backup utilizando LZW para compactação e recuperação de dados, além de implementar uma lista invertida, árvore B+ e tabela hash para organização e indexação eficientes. Estas estruturas garantem operações rápidas e acesso direto aos dados, tornando o sistema robusto e completo, com funcionalidades de backup e recuperação que permitem ao usuário escolher versões específicas para restauração.
